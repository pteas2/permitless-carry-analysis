# Firearm Carrying and Purchase Intentions in Permitless Carry States

Replication code for:

**Teas, P.E., Rozario, T.D., & Filindra, A. (2026). Firearm Carrying and Purchase Intentions in Permitless Carry States: State Policy or State Culture?**

Preprint: https://doi.org/10.31234/osf.io/asxjv_v1

## Overview

This study examines associations between state permitless carry laws and individual-level firearm behaviors using a nationally representative survey of 18,251 U.S. adults (2024-2025). We find that while residents of permitless carry states report higher rates of gun ownership, carrying, and purchase intentions, these associations largely disappear after accounting for state-level political context (Trump vote share) and gun-ownership rates.

## Repository Contents

This repository contains:
1. **Weight creation script** (`weight_creation_github.Rmd`) - Documents the survey raking procedure used to generate post-stratification weights
2. **Main analysis script** (`everytown_analyses_V3_github.Rmd`) - Contains all models, tables, and figures from the manuscript

**Note:** Survey microdata and intermediate datasets are **not included** in this repository due to potentially identifying information in the responses. The code is provided for transparency and methodological review.

## Survey Weighting Methodology

The survey data (N=18,586 pre-exclusions) were collected via Lucid-Cint, an opt-in online panel. Post-stratification weights were generated using iterative proportional fitting (raking) to adjust the sample to match national population targets.

**Raking variables:**
- Gender (2 categories)
- Age (4 categories: 18-29, 30-44, 45-64, 65+)
- Education (4 categories: HS or less, Some college, College, Graduate)
- Household income (5 categories)
- Party identification with leaners (3 categories: Democrat, Independent, Republican)
- Race × Region interaction (16 categories)

**Population targets:**
- Gender, age, education, race × region: 2024 ACS PUMS 1-year estimates
- Income: 2024 Census CPS HINC-06 table
- Party ID: Gallup 2024 party affiliation (with leaners)

**Weight trimming:** Floor = 0.2, cap = 7.0 (normalized to mean = 1)

See `weight_creation_github.Rmd` for complete implementation details and comparison with Stata IPF implementation.

## Reproducibility Notes

**Data Availability:** Due to potentially identifying information in survey responses, the underlying datasets are not included in this repository. The code is provided for:
- Methodological transparency and peer review
- Documentation of analytical decisions
- Reference for researchers conducting similar work

**If data were available**, the workflow would be:
1. Run `weight_creation_github.Rmd` to generate post-stratification weights
2. Run `everytown_analyses_V3_github.Rmd` sequentially through "Run everything above here" to set up the environment
3. Individual analysis chunks can then be run independently

**Key methodological details:**
- Survey weights (`wgtpi`) are used for full-sample analyses; gun-owner-only analyses are unweighted
- Weights generated using iterative proportional fitting (raking) on 6 dimensions: gender, age, education, income, party ID, and race × region
- All models use robust standard errors clustered by state
- Weight trimming: floor = 0.2, cap = 7.0
